# Backrooms Architecture Design Guide

> **프로젝트 최상위 설계 기준 문서 (v2.0)**  
> Level 0 클래식 백룸 레퍼런스 **10장**을 건축학적 관점에서 교차 분석한 결과입니다.  
> 본 문서는 **던전 생성 규칙이 아니라**, 오래된 사무실·상업 층을 절차적으로 재현하기 위한 건축 설계 기준입니다.

**분석 대상:** 레퍼런스 10장 (2회에 걸쳐 제공된 이미지 세트 통합·중복 제거)  
**분석 방법:** 이미지별 특징 추출 → 10장 교차 비교 → **2장 이상 공통**만 채택 → 1장 한정 특징 제외  
**코드 상태:** 본 문서 작성·개정 시점에 **게임 코드는 수정하지 않음**

### 레퍼런스 10장 역할 분류

| ID | 유형 | 핵심 특징 |
|----|------|-----------|
| REF-01 | 대형 오픈 라운지 | 넓은 바닥, 파티션만 존재, 시야 다방향 개방 |
| REF-02 | 라운지 + 개구부 전환 | 어두운 전경 → 밝은 후경, 넓은 무틀 개구부 |
| REF-03 | 자립형 파티션 배치 | 벽이 실처럼 배치, L/T 코너, 걸레받이·콘센트 |
| REF-04 | **긴 복도 (예외)** | 1점 투시 복도, 폭 3–4m, 측면 무틀 개구부 |
| REF-05 | 두꺼운 블록 파티션 | 0.5–1m급 덩어리 벽, 개방 연결, 몰딩 없음 |
| REF-06 | 두꺼운 블록 + 다마스크 | 구조적 블록 + 미세 플로럴 패턴, 걸레받이 |
| REF-07 | 비선형 직교 미로 | 자유 배치 파티션, end cap, 15–20cm 두께 |
| REF-08 | 반파티션·기둥 혼합 | 허리 높이 파티션, 사각 기둥 (희귀 변형) |
| REF-09 | 외부 코너 래핑 | 넓은 라운지, 외부 모서리 벽지 연속 강조 |
| REF-10 | 복도형 통로 연속 | 바닥·천장 끊김 없이 이어지는 통로 + 측실 개구 |

## 0. Executive Summary

백룸 Level 0의 본질:

> **「좁은 복도 미로」가 아니라 「넓은 사무 층 바닥 위에 경량 파티션이 놓인 공간」**

| 항목 | 백룸 (10장 공통) | 전형적 던전 (지양) |
|------|------------------|-------------------|
| 중심 공간 | 라운지·오픈 플로어 (9/10) | 복도 + 작은 방 |
| 연결 | 개방형 개구부, 문 없음 (10/10) | 문·게이트·좁은 통로 |
| 벽 역할 | 시선·동선 분할 파티션 | 미로 외벽 |
| 시야 | 다구역 동시 가시 (8/10) | 단방향 제한 |
| 바닥·천장 | 청크 전체 연속 (9/10) | 방마다 단절 |
| 벽지 | 월드 연속, 수직 패턴 (10/10) | 면별 타일 |

## 1. 방과 복도의 비율

### 1.1 정량적 결론 (10장 교차)

| 공간 유형 | 등장 장수 | 비율 | 판정 |
|-----------|-----------|------|------|
| 대형 오픈 라운지·오픈 플로어 | **9/10** | ~90% | **지배적** |
| 중간 폭 통로 (벽 사이 3–6m) | 8/10 | ~80% | 흔함 |
| 긴 1차원 복도가 **주역** | **1/10** | ~10% | **예외** |
| 좁은 복도 (<2.5m) | 0/10 | 0% | **없음** |

**핵심:** 복도는 존재하지만 **공간의 주인공이 아니다**. 10장 중 9장은 「방」보다 「구역(zone)」이 중심이며, 복도는 라운지 사이에 **자연스럽게 남는 넓은 통로** 수준이다.

### 1.2 「방」과 「복도」의 관계

10장 모두에서 공통:

- **방/복도 이분법이 성립하지 않음** — 기능적 실(room)이 아니라 **바닥 연속체 위의 구역**
- 벽은 공간을 **완전히 밀폐하지 않음** — 1~3면이 열린 파티션이 대다수
- 동선은 **그래프(tree)가 아닌 평면 파티션 배치**로 형성

### 1.3 공간 체적 비율 (생성 목표)

```
넓은 개방 구역 (라운지·오픈 플로어)     65–80%
중간 통로 (폭 3–6m)                    15–25%
긴 복도형 띠 (폭 ≥3m, 길이/폭 > 3)      ≤10%
좁은 복도형 (폭 < 2.5m)                  0%
```

### 1.4 공간 연결 리듬

10장에서 반복되는 동선 패턴:

```
[넓은 라운지] ──개구부──▶ [중간 구역] ──개구부──▶ [또 다른 라운지]
       │                           │
       └── 파티션(부분 높이/전체) ──┘
```

- **「방 → 방 → 넓은 공간 → 방 → 방」** 리듬이 8/10에서 확인
- 한 지점에서 **2개 이상 구역이 동시에 보임** (7/10)
- 조명 대비(어두운 전경 → 밝은 후경)는 2/10 — 분위기 연출용, 필수 아님

## 2. 방의 형태

### 2.1 가장 많이 등장하는 형태 (빈도순)

| 순위 | 형태 | 등장 | 설명 |
|------|------|------|------|
| 1 | **직사각 오픈 존** | 10/10 | 가장 기본 단위 |
| 2 | **L자 구역** | 9/10 | 벽 꺾임으로 자연 발생 |
| 3 | **T자 교차** | 8/10 | 파티션 끝이 다른 벽과 만남 |
| 4 | **자립형 파티션 핀(fin)** | 7/10 | 한쪽 끝이 열린 벽 섬 |
| 5 | **ㄷ·U자 알코브** | 5/10 | 세 면이 막힌 움푹한 공간 |
| 6 | ㄹ자·지그재그 | 3/10 | 비선형 동선 유도 |

### 2.2 방 크기 분포

천장고 **2.7–3.0m** 기준 (10/10 공통):

| 유형 | 대략 크기 | 등장 | 비고 |
|------|-----------|------|------|
| 소형 알코브 | 3×3 ~ 4×5m | 6/10 | 파티션으로 생긴 움푹한 곳 |
| 중형 구역 | 5×6 ~ 8×10m | **10/10** | 가장 흔한 체류 공간 |
| 대형 라운지 | 10×12m+ | 8/10 | 시야·조명이 열림 |
| 긴 축 공간 | 한 축 12m+, 폭 ≥4m | 4/10 | 「긴 방」이지 좁은 복도가 아님 |

**크기 분포 목표:** 소 : 중 : 대 ≈ **2 : 5 : 3**

### 2.3 긴 방 vs 넓은 방

- **넓은 방(양축 모두 큼)** >> **긴 띠형 공간** (9/10)
- 가로:세로 비율 대부분 **1:1 ~ 1:2.5**
- **1:4 이상의 좁은 띠** — 복도 예외(REF-04) 외 **없음**
- 긴 공간이 있어도 **폭 최소 ~3–4m** 유지 (10/10)

### 2.4 직교 구조 특징

**10/10 공통:**

- **100% 직교(90°)** — 대각선·곡선·경사 없음
- 모든 요소가 **공통 격자에 정렬** (벽·천장 타일·조명)
- 완전 폐실(四门)보다 **열린 파티션**이 지배적 (9/10)

**1장 한정 (생성 필수 아님):**

| 특징 | 등장 | 처리 |
|------|------|------|
| 허리 높이 반파티션 (~1m) | 2/10 | 선택 변형 |
| 무릎 높이 범퍼 레일 | 1/10 | **제외** |
| 리벳 구조 기둥 | 1/10 | **제외** |
| 0.5–1m 초두꺼운 블록 벽 | 3/10 | 스타일 변형, 기본값 아님 |

## 3. 공간 연결 방식

### 3.1 연결 메커니즘 (10/10 공통)

| 방식 | 등장 | 설명 |
|------|------|------|
| **문 없음** | **10/10** | 문짝·손잡이·문틀 프레임 없음 |
| **개방형 개구부** | 10/10 | 벽이 끝나거나 제거되어 생기는 직사각 opening |
| **바닥·천장 연속** | 9/10 | 구역 간 높이·재질 단절 없음 |
| **벽 두께 면 노출** | 9/10 | 개구부 가장자리에서 단면이 보임 |

### 3.2 「문」의 부재

10장 **모두**에서 확인:

```
✗ 문짝(door leaf)
✗ 문틀(frame/casing)
✗ 문지방(threshold) 단차
✗ 힌지·손잡이
```

연결은 **벽의 subtractive opening(제거형 개구)** 로만 표현된다.

### 3.3 개구부 특성

| 속성 | 관측값 | 등장 |
|------|--------|------|
| 최소 폭 | ~2m 이상 | 9/10 |
| 이상적 폭 | 2.5–6m (벽 폭의 30–70%) | 8/10 |
| 형태 | 직사각, floor-to-ceiling | 10/10 |
| 프레임 | 없음 | 10/10 |

### 3.4 긴 복도의 실제 등장

| 질문 | 답 |
|------|-----|
| 긴 복도가 있나? | **있음** — REF-04, REF-10 |
| 주 공간인가? | **아님** — 10장 중 2장 (20%) |
| 폭은? | **3–4m** — 좁은 미로 복도가 아님 |
| 측면 연결? | 복도 벽에 **무틀 개구부**로 측 구역 연결 |
| 생성기 기본값? | **복도 우선 생성 금지** |

### 3.5 연결 규칙 요약

```
✓ OPENING_ONLY — 개구부만
✓ MIN_OPENING_WIDTH ≥ 2.0m
✓ FLOOR_CONTINUOUS — 바닥 재질·높이 연속
✓ CEILING_CONTINUOUS — 천장 격자 연속
✗ DOOR_MESH / DOOR_FRAME
✗ 1m급 게이트
✗ 일방통행 시퀀스 방
```

## 4. 건축 특징

### 4.1 벽 두께

10장에서 **두 가지 시각적 계층**이 관측됨:

| 유형 | 두께 | 등장 | 채택 |
|------|------|------|------|
| **표준 경량 파티션** | 10–20cm | **7/10** | **생성 기본값** |
| 두꺼운 블록/기둥형 | 50–100cm | 3/10 | 스타일 변형, 비권장 |

**생성 기준: 16cm (0.16m)** — 7/10 다수 레퍼런스와 현실 건식벽 일치.

**10/10 공통 규칙:**

- 코너에서 **두께 증가·감소 없음**
- 개구부 잼브에서 **동일 두께 유지**
- end cap(벽 끝 단면) **명확히 존재** (9/10)

### 4.2 코너 처리

**10/10 공통:**

| 항목 | 관측 |
|------|------|
| 각도 | **90° 날카로운 직각** |
| 틈(gap) | **없음** |
| 돌출(pillar) | **없음** (올바른 레퍼런스) |
| 내부 코너 | 패턴·재질 연속 |
| 외부 코너 | 패턴 wrap 연속 (9/10) |
| end cap | 동일 재질·패턴 (9/10) |

### 4.3 천장 구조

**10/10 공통:**

- **매립형 천장(drop / suspended ceiling)**
- **사각 격자 타일** (어쿠스틱 패널)
- 격자선이 **벽 축과 평행** 정렬
- 천장고 **2.7–3.0m** (인체 스케일 기준)

### 4.4 조명 배치

**10/10 공통:**

| 요소 | 관측 |
|------|------|
| 광원 | **형광 트로퍼(troffer)** — 천장 격자 매립 직사각 패널 |
| 색온도 | 따뜻한 **황록·담황** 인공광 |
| 배치 | 격자 스냅, 행(row) 또는 격자 패턴 |
| 규칙성 | 완벽 단일 격자보다 **약간 불규칙** (7/10) — 기계적 완벽함보다 사무실스러운 반복 |
| 그림자 | **부드럽고 확산** — 극단 대비 없음 |
| 자연광 | **없음** (10/10) |

**2/10에서만:** 전경 어두움 → 후경 밝음 대비 (분위기 연출, 필수 아님)

### 4.5 몰딩 유무

10장 교차 결과 — **분열 항목**:

| 유형 | 등장 | 결론 |
|------|------|------|
| 걸레받이(baseboard) | **3/10** | 소수 |
| 천장 몰딩(crown) | 0/10 | **없음** |
| 문 케이싱 | 0/10 | **없음** |
| 반파티션 상단 캡 | 2/10 | 희귀 변형 |
| 무릎 높이 레일 | 1/10 | **제외** |

**생성 기준:** 기본 **몰딩 없음** (7/10이 날카로운 벽-바닥·벽-천장 직접 접합).  
걸레받이는 **선택 변형**으로만 허용 (확률 ≤20%).

### 4.6 현실 건축물과의 공통점

10/10 공통으로 대응되는 현실 유형:

- **1970–1990년대 미국·유럽 상업 오피스 층**
- **건식 경량벽(drywall partition)** + 이동식 칸막이 배치
- **개방형 사무실(open-plan)** 의 파편화 — 실별 완전 분리 아님
- **창문 없음** — 내부 코어·비현실적 무한 층
- **저가 인테리어** — 단일 카펫, 단일 벽지, 표준 드롭 천장
- **사용 중단된 공간** — 가구 없거나 산발적 (가구는 건축 규칙 아님)

## 5. 벽지 분석

### 5.1 패턴 특성 (10/10 공통)

| 속성 | 관측 |
|------|------|
| 색상 | **담황·베이지·황록** 단색조, 저채도 |
| 방향 | **수직(vertical)** — 중력(Y)축 정렬 |
| 패턴 유형 | 미세 세로 줄무늬 / chevron(↑) / 다마스크 / 스티플 노이즈 |
| 대비 | **저대비** — 멀리서면 거의 단색 |
| 반복 | 작고 균일, 눈에 띄는 대형 프린트 없음 |

### 5.2 UV 연속 방식 (10/10 — 최우선 시각 규칙)

```
UV_MODE = WORLD_SPACE (또는 동등한 연속 투영)
PATTERN_AXIS = WORLD_Y_ALIGNED
```

- 개별 벽 면·박스마다 **독립 타일링 하지 않음**
- **모든 면에서 세로 패턴이 수직 유지**
- 청크·벽 세그먼트 경계를 넘어 **패턴 연속**
- 스케일 **전 구역 동일** — 면마다 zoom in/out 없음

### 5.3 코너에서의 패턴 (10/10 공통)

| 코너 유형 | 관측 |
|-----------|------|
| **내부 코너(concave)** | 패턴이 꺾이며 이어짐, 끊김·회전·스케일 변화 없음 |
| **외부 코너(convex)** | 패턴이 모서리를 **감싸 wrap**, REF-09에서 특히 명확 |
| **T/십자 접합** | 패턴 단절 없음 |
| 금지 사항 | 90° UV 강제 회전, 코너용 별도 텍스처, 회색 캡 |

### 5.4 벽 두께·단면·잼브 재질 (10/10 공통)

| 면 | 재질 |
|----|------|
| 벽 전면 | 동일 벽지 |
| 벽 후면 | 동일 벽지 |
| **벽 단면(end cap)** | **동일 벽지**, 패턴 정렬 |
| **개구부 잼브** | **동일 벽지** |
| 두께 면 단색 캡 | **없음** (10/10) |

### 5.5 벽지 금지 사항


## 7. 생성 규칙 (Procedural Office Generator Rules)

### 7.1 공간 비율 예산 (Space Budget)

```yaml
ROOM_FLOOR_TARGET:     ≥ 80%    # 바닥 면적 중 체류 가능 존
CORRIDOR_LIKE_MAX:     ≤ 18%    # 좁은 통로형 셀
LOUNGE_MIN_COUNT:      ≥ 1      # 청크당 최소 1개 대형 존
LOUNGE_MIN_AREA:       ≥ 30%    # 청크 바닥 면적
MIN_OPENING_WIDTH:     ≥ 2.0m
PREFERRED_OPENING:     ≥ 2.5m
MIN_PASSAGE_WIDTH:     ≥ 3.0m   # 통로 최소 폭
NARROW_PASSAGE_BAN:    < 2.5m   # 생성 금지
```

### 7.2 존(Zone) 배치

1. **시드 라운지** — 무게중심에 가장 큰 존 배치
2. **변 공유 부착** — 새 존은 기존 바닥과 **edge-to-edge** 접합 (겹침 없음)
3. **형태 가중치:**

| 형태 | 가중치 |
|------|--------|
| 직사각 | 30% |
| L자 | 25% |
| 대형 라운지 | 20% |
| 긴 직사각 (폭≥4m) | 10% |
| ㄷ·ㄹ·T 변형 | 10% |
| 자립형 파티션 핀 | 5% |

4. **폭 < 3m인 띠형 존 생성 금지** — 복도화 방지

### 7.3 연결 규칙

```yaml
CONNECTION_MODE:        OPENING_ONLY
OPENING_IN_PARTITION:   true       # 공유 파티션 벽 제거
CHUNK_EDGE_OPENING:     true       # 청크 경계도 개구부
DOOR_MESH:              false      # 문짝 없음
DOOR_FRAME:             false      # 문틀 없음
MIN_CONNECTIONS_PER_ZONE: 1
EXTRA_CONNECTIONS:      30–45%     # MST 외 추가 루프
```

- 존 간 연결 = **공유 벽의 subtractive gap** (벽 세그먼트 부재)
- 완전 미로가 아닌 **여러 경로 가능한 사무 층**

### 7.4 벽 생성 규칙

```yaml
WALL_THICKNESS:         0.16m      # 전 구간 일정
WALL_MESH_MODE:         CONTINUOUS_FOOTPRINT
CORNER_MODE:            IMPLICIT_UNION
BOX_STITCHING:          false      # BoxGeometry 이어붙이기 금지
CORNER_BOX_OVERLAY:     false      # 코너 보정 박스 금지
CONNECTOR_ANALYSIS:     true
```

**Connector 분류 (기하학적 연속체의 일부, 별도 오브젝트 아님):**

| 유형 | 조건 |
|------|------|
| Straight Wall | 정점 연결 2, 180° |
| End Cap | 정점 연결 1 |
| Inner Corner | 연결 2, 90°, walkable 1 quadrant |
| Outer Corner | 연결 2, 90°, walkable 3 quadrants |
| T Junction | 연결 3 |
| Cross Junction | 연결 4 |
| Door Opening | colinear gap 0.9–3.2m |
| Wide Opening | colinear gap ≥3.2m |

**생성 후 필수 검증:**

```
protrusion_cells     = 0
corner_thickness_drift ≤ 1mm
dangling_segments    = 0
jamb_faces_missing   = 0
uv_seam_at_corner    = 0
```

### 7.5 벽지·UV 규칙

```yaml
UV_MODE:              WORLD_SPACE
PATTERN_AXIS:         VERTICAL
WALL_MATERIAL:        SINGLE_WALLPAPER
THICKNESS_FACES:      SAME_MATERIAL
JAMB_FACES:           SAME_MATERIAL
END_CAP_FACES:        SAME_MATERIAL
TILE_WIDTH:           0.76m     # 사용자 제공 벽지
CHUNK_UV_CONTINUOUS:  true      # roomWx/roomWz 오프셋
```

- U축: 면 법선 기준 `worldX` 또는 `worldZ` / tileW
- V축: `worldY` / tileH
- 코너: **정점 공유** + 동일 UV 투영

### 7.6 천장·바닥·조명

| 요소 | 규칙 |
|------|------|
| 바닥 | **walkable 존만** 카펫 (10/10 연속 바닥 — 현재 Gap) |
| 천장 | 드롭 타일 격자, PANEL_SIZE 스냅, 벽 축 정렬 |
| 조명 | 트로퍼 인스턴스, 격자 스냅, 황색광, 행/격자 배치 |
| 몰딩 | 기본 없음, 걸레받이 ≤20% 확률 |

### 7.7 검증 파이프라인

```
1. Grid connectivity flood    → 단일 바닥 컴포넌트
2. Passage width check        → 모든 개구부 ≥ MIN_OPENING_WIDTH
3. Room/corridor ratio        → corridor ≤ 18%, lounge ≥ 30%
4. Wall footprint QA          → protrusion, gap, thickness
5. Connector classification   → 모든 정점 타입 기록
6. Nav flood from chunk exits → 4방향 출구 연결
7. UV continuity spot-check  → 코너 정점 UV 정렬
```

실패 시: **재생성 (max 12) → repair pass → fallback 단일 라운지**

## 9. 용어 정의

| 용어 | 정의 |
|------|------|
| **Zone** | 바닥 패킹의 연결된 직교 영역. 완전 폐실 아님 |
| **Lounge** | 면적이 큰 zone, 동선·시야의 허브 |
| **Opening** | 파티션 벽이 없는 연결 — 문이 아님 |
| **Partition** | 바닥~천장 경량 내벽 |
| **Fin** | 한쪽 이상 열린 자립형 파티션 |
| **Connector** | 벽 정점의 기하학적 연결 유형 — 별도 mesh 아님 |
| **Corridor-like** | 폭 <3m & 길이/폭 >2.5 인 띠형 공간 |
| **Jamb** | 개구부 측면 두께 면 |

## 10. 설계 원칙 (한 문장)

> **「1980년대 사무 층을 한 장의 연속 벽지로 시공한 뒤, 가구만 치운 무한 층」을 만든다.**

- 겹친 벽 ❌
- 코너 틈 ❌
- 벽 두께 오류 ❌
- 벽지 패턴 끊김 ❌
- UV 깨짐 ❌
- 문틀·문짝 ❌
- 1m급 좁은 복도 ❌
